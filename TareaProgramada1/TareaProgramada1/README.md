   3.2-Modo consola explícito

   3.3Modo GUI (JOptionPane)
      java Main -gui

## Instrucciones de uso

   1. Al iniciar, aparece el menú principal con 3 opciones.
   2. Opción 1: Ver el historial del mejor equipo registrado.
   3. Opción 2: Iniciar una nueva partida — ingresar nombre de equipo y de cada jugador.
   4. Durante la partida, ver el estado del juego y realizar combos seleccionando:
      - Qué carta de cada jugador va contra cada carta del oponente.
      - La cantidad de ataques del combo.
   5. Al finalizar se revelan todos los atributos de cartas y oponentes.
   6. Opción 3: Salir del programa.
